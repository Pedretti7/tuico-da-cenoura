# Tuico da Cenoura

Jogo mobile em HTML5 do Tuico mordendo uma cenoura gigante e desbloqueando trajes por recordes.

## Jogar

Quando o GitHub Pages estiver ativado, abra:

https://pedretti7.github.io/tuico-da-cenoura/

No iPhone, abra o link pelo Safari, toque em compartilhar e escolha **Adicionar à Tela de Início**.

## Observação técnica

As skins foram publicadas como arquivos texto `.webp.b64` para facilitar o upload pelo GitHub. O jogo reconstrói as imagens automaticamente no navegador.
